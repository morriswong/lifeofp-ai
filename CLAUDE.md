# Workspace Rules

This is Morris's default Cowork folder, synced across devices. Keep it organized.

## Folder Structure

```
writing/          — M&A newsletter system, style guides, CHreading, prev-work (blog posts)
career/           — Resume versions, job applications (coindesk, coinbase dashboards)
projects/
  finance/        — BRK portfolio tracker
  travel/         — Okinawa trip planner
  wedding/        — Wedding budget, plan, session summary
setup/            — Claude browser scripts, quickstart guides
.scratch/         — Ephemeral session data (calendar snapshots, agent shaping)
```

## File Placement Rule

**No loose files at the root.** When creating any new file or output, always use `AskUserQuestion` to ask Morris which folder it belongs in before saving. If none of the existing folders fit, propose a new subfolder name for approval.

This applies to every deliverable — documents, spreadsheets, dashboards, scripts, etc. The only exceptions are dotfiles and config files (like this one).
