## 【模板 D：引用格式】

### 文中引用範例

**數據引用：**
```
「商譽減損達200億美元」
（來源：Vodafone Annual Report 2001, p.47）
```

**引言引用：**
```
「Gent在內部會議中承認『時間不站在我們這邊』」
（來源：Financial Times專訪，2005-03-15）
```

**分析性內容標示：**
```
「這可能意味著Vodafone當時面臨的壓力遠大於他們願意承認的。」
（分析依據：內部郵件洩露 + 董事會會議記錄）
```

**場景重建標示：**
```
「據參與者回憶，那天晚上的會議氣氛極度緊張...」
（來源：Horn-Smith回憶錄，2005）
```
