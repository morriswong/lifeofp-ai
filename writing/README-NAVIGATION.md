## 💡 核心理念（記住這個！）

### 從 newsletter-brief-ENHANCED.md 提煉的精髓：

> **用朋友般親切的語氣，講述資深分析師才懂的故事，**
> **讓讀者在不知不覺中理解商業世界的殘酷與荒謬。**

**三個平衡**：
1. **親和力 ↔ 犀利度**：輕鬆但不膚淺，批判但不憤世嫉俗
2. **故事性 ↔ 專業性**：有戲劇張力，但所有事實可驗證
3. **深度 ↔ 可讀性**：挖掘深層動機，但用白話文講

**雙重語調融合**：
- 從 `writing-style-prompt.md` 繼承：親和、誠實、可讀
- 從 `newsletter-brief.md` 保留：批判、深度、故事性

## 🆘 遇到困難時

### 如果寫得很痛苦...
```
1. 檢查：是不是資料不夠？
   → 回到 Phase 2 補充研究

2. 檢查：是不是想太完美？
   → 記住：初稿就是要醜，重點是寫完

3. 檢查：是不是太在意字數？
   → 先寫完整個故事，再調整字數
```

### 如果不確定質量...
```
用 newsletter-brief-ENHANCED.md > 第十三章 > D. 完稿前最後確認

9個核心問題：
□ 字數3000-5000字？
□ 有近期鉤子連接歷史？
□ 所有數字有來源？
□ 語調親和但犀利？
□ 有1-2個「金句」？
□ 有讓人印象深刻的場景？
□ 文末有參考資料列表？
□ 檢查過禁用詞？
□ 讀起來像「和朋友聊天」而非「寫報告」？
```

### 如果缺乏靈感...
```
1. 翻閱 newsletter-brief.md > 第十章「範例對比總結」
2. 看看別人的M&A分析（但別抄襲，要建立自己觀點）
3. 用 article-production-templates.md > 模板G 複習過去的金句
4. 休息一下，明天再寫 :)
```
